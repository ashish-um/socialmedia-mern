## Features
1. Authentication (Login, Registration)
2. Create Posts
3. Like Other Posts
4. Save Posts

## Tech Stack

- **Frontend:**
  - React.js
  - Axios (for API calls)
  - React Cookie(for Autorization)
  - Custom UI with vanilla css

- **Backend:**
  - Node.js
  - Express.js
  - MongoDB (with Mongoose for ODM)
  - JWT (for authentication)
